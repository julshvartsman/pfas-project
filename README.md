## Installation Guide
1. Clone the application from the repo. Navigate in your terminal to your desired system directory for the project and run `git clone https://github.com/julshvartsman/pfas-project.git`. This will create a new directory called `pfas-project`.
2. Navigate into the `pfas-project` folder.
3. Set up a python virtual environment. From the top level of `pfas-project` run `python -m venv env` to create a new virtual environment. This may take a few moments.
4. Activate your virtual environment via the activation script. On Linux and MacOS, the command is `source env/bin/activate`. If you're on Windows, the correct command may be different; consult the table on [this](https://docs.python.org/3/library/venv.html) documentation page for details.
5. With your virtual environment active (this is indicated by a little `(env)` next to your username in the terminal) run the command `pip install -r requirements.txt` to install all dependencies listed in the `requirements.txt` file. This will take a little while to run.
6. All dependencies are now installed and you can navigate to your preferred IDE or workspace to contribute. For VS code, the command is `code .` and for Jupyter lab, `jupyter lab`.
